## 2.Java常用数学函数：  
	·Java Math.round() 四舍五入  
	·Java Math.random() 获取随机数  
	·java Math.pow()计算次方  
	·Java Math.ceil() 取整  
	·Java Math.sqrt() 计算平方根  
	·Java Math.log() 计算对数  
	·Java Math.floor() 取整  
	·Java Math.toDegrees()弧度转换成角度  
	·Java Math.toRadians()角度转换为弧度  
	·Java Math.cos() 计算余弦  
	·Java Math.acos() 计算反余弦p

	·Java Math.asin() 计算反正弦  
	·Java Math.tan()/Math.tanh() 计算正切/双曲正切
